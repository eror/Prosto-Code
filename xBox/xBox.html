System requirements

The Lucom Interaction Platform demands an operating system for which the following components are available:

  1. A servlet 2.4/JSP 2.0 interoperable servlet engine, e.g. a J2EE compliant application server (same servlet/JSP version number)
  2. Java Development Kit (JDK) 5.0 or higher
  3. An SQL database with a JDBC 2.0 compliant driver
